## 原型及原型链

### 在原型上添加属性或者⽅法有什什么好处?

如果不不通过原型的⽅方式，每生成⼀一个新对象，都会在内存中新开辟⼀一块存储 空间，当对象变多之后，性能会变得很差。

通过

```js
Player.prototype.xx = function () {};
Player.prototype.xx = function () {};
Player.prototype.xx = function () {};

```
这种⽅方式向原型对象添加属性或者⽅方法的话，⼜又显得⾮非常麻烦。所以我们可以这样写

```js
Player.prototype = {
  start: function () {
    console.log("下棋");
   },
  revert: function () {
    console.log("悔悔棋");
  },
}
```
